Garden MLOps: Automated Vegetation Monitoring Pipeline

Developed an automated MLOps pipeline that monitors garden health using Raspberry Pi camera integration and serverless cloud infrastructure. The system captures images at regular intervals, uploads to Google Cloud Storage, and triggers Cloud Functions to analyze vegetation health via green pixel ratio analysis. Implemented Infrastructure as Code with Terraform for reproducible GCP resource provisioning, with automated alerting for low-vegetation conditions. Technologies: Python, Google Cloud Functions, Cloud Storage, Terraform, Raspberry Pi, Pillow.
